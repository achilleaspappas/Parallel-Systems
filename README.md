# Parallel Systems

A university project for openMP in C

## Requirements

- [WSL2 for Windows](https://docs.microsoft.com/en-us/windows/wsl/install) or any Linux distribution 
- [GCC](https://gcc.gnu.org/)

You can install GCC
```
sudo apt install gcc
```

Compile openMP code
```
gcc file.c -o file -fopenmp
```

Execute openMP code 
```
./file
```
